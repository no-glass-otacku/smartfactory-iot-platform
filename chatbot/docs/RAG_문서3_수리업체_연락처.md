## 수리 의뢰 절차

### 평일 주간 (09:00~18:00)
1. 담당 업체 주 담당자에게 전화 연락
2. 설비 ID, 고장 유형, 현재 증상 구두 설명
3. 출동 예정 시간 확인 후 설비팀장에게 보고
4. 수리 완료 후 작업 확인서 수령 및 수리 이력 시스템 입력

### 야간·주말 긴급 (18:00 이후, 주말·공휴일)
1. 담당 업체 긴급 연락처로 즉시 전화
2. 클린룸 영향 여부를 함께 설명 (온도 편차 발생 여부)
3. 출동 예정 시간 확인 후 설비팀장 및 생산관리자 동시 보고
4. 긴급 출동 시 보안실에 외부 업체 출입 사전 등록 필수 (보안실: 내선 100)

### 수리 의뢰 시 필수 전달 정보
- 설비 ID (예: M-014)
- 고장 유형 (시스템 알림 내용 그대로 전달)
- 현재 센서값 (온도, 진동, 압력 등)
- 마지막 정상 운전 시각
- 클린룸 환경 영향 여부
