# 공조 압축기 정상 상태 정기 점검 가이드라인 (Normal)

**문서 번호**: MNT-GUIDE-005  
**최종 수정일**: 2024-11-15  
**적용 설비**: 클린룸 A동 공조용 스크류 압축기 (M-001 ~ M-030)  
**작성 부서**: 설비관리팀

---

## 1. 정상 상태 판정 기준

시스템에서 아래 조건을 모두 충족하면 Normal(정상)으로 판정한다.

| 항목 | 정상 범위 |
|---|---|
| 토출 온도 | 70~88°C |
| 흡입 압력 | 3.5~4.5 bar |
| 토출 압력 | 7.5~8.5 bar |
| 진동 수치 | 10~35 |
| 운전 전류 | 정격 전류의 70~100% |
| 오일 레벨 | 1/4 ~ 3/4 범위 |
| 클린룸 온도 | 23 ± 0.5°C |
| 클린룸 습도 | 45 ± 5% RH |
| 클린룸 차압 | +12.5 Pa 이상 |
| anomaly_flag | 0 |
| maintenance_required | 0 |

> 정상 상태에서도 정기 점검 일정에 따라 예방 정비를 실시한다. 예방 정비를 통해 고장을 사전에 방지하고 클린룸 환경 안정성을 유지한다.

---

## 2. 정기 점검 일정

### 일간 점검 (매일 1회, 주간 08:00 / 야간 20:00)

| 점검 항목 | 점검 방법 | 기준값 |
|---|---|---|
| 운전 패널 이상 표시 확인 | 육안 | 이상 표시 없음 |
| 토출 온도 확인 | 운전 패널 | 70~88°C |
| 토출 압력 확인 | 운전 패널 | 7.5~8.5 bar |
| 진동·소음 이상 유무 | 청각·촉각 | 이상 없음 |
| 오일 레벨 확인 | 사이트 글라스 | 1/4 ~ 3/4 |
| 클린룸 온·습도 확인 | 모니터링 패널 | 온도 23±0.5°C, 습도 45±5% RH |
| 클린룸 차압 확인 | 차압 게이지 | +12.5 Pa 이상 |
| 냉각수 누수 육안 점검 | 육안 | 누수 없음 |

### 주간 점검 (매주 월요일 09:00, 설비관리팀)

| 점검 항목 | 점검 방법 | 기준값 |
|---|---|---|
| 냉각 팬 동작 상태 | 육안·청각 | 정상 회전, 이상 소음 없음 |
| 인버터 디스플레이 에러 이력 | 인버터 패널 조회 | 에러 이력 없음 |
| 운전 전류 측정 | 클램프 미터 | 정격 전류의 70~100% |
| 흡입 필터 차압 확인 | 차압 게이지 | 0.5 kPa 이하 |
| 유분리기 차압 확인 | 차압 게이지 | 0.1 MPa 이하 |
| 제어반 내부 온도 확인 | 운전 패널 | 45°C 이하 |
| 오일 색상·상태 확인 | 사이트 글라스 | 투명~연황색 |

### 월간 점검 (매월 첫째 주 금요일, 설비관리팀 + 외부 업체)

| 점검 항목 | 점검 방법 | 기준값 |
|---|---|---|
| 냉매 압력 측정 | 매니폴드 게이지 | 흡입 3.5~4.5 bar, 토출 7.5~8.5 bar |
| 오일 레벨·색상·점도 | 채취 후 육안 및 측정 | 투명~연황색, 정상 점도 |
| 모터 운전 전류 3상 측정 | 클램프 미터 | 불평형 2% 이내 |
| 베어링 온도 측정 | 접촉식 온도계 | 주변 온도 + 40°C 이하 |
| 냉각 팬 벨트 장력 확인 | 장력 측정기 | 제조사 기준값 ± 10% |
| 흡입 필터 청소 또는 교체 | 육안·차압 기준 | 차압 0.5 kPa 이하 유지 |
| 배관 연결부 누설 점검 | 누설 감지기 | 누설 없음 |
| 제어반 단자 볼트 체결 확인 | 토크렌치 | 제조사 기준 토크 |

### 분기 점검 (3월·6월·9월·12월 첫째 주, 외부 업체 필수 참여)

| 점검 항목 | 점검 방법 | 기준값 |
|---|---|---|
| 오일 분석 (점도, 산가, 수분) | 채취 후 분석 의뢰 | 제조사 기준 이내 |
| 모터 절연 저항 측정 | 절연 저항계 | 1 MΩ 이상 |
| 인버터 파라미터 점검 | 인버터 설정값 확인 | 설계 기준값 일치 |
| 안전 밸브 동작 압력 확인 | 수동 동작 시험 | 설정 압력 ± 3% |
| 방진 패드 상태 점검 | 육안 | 균열·변형 없음 |
| 냉매 충전량 확인 | 사이트 글라스·압력 | 기포 없음, 정상 압력 |
| 전반적인 성능 테스트 | 부하 운전 중 측정 | 설계 성능의 95% 이상 |

### 연간 점검 (매년 1월, 제조사 AS 또는 전문 업체)

| 점검 항목 | 내용 |
|---|---|
| 압축기 주요 부품 분해 점검 | 스크류 로터, 베어링, 씰 마모 상태 |
| 오일 전량 교환 | 제조사 권장 오일 사용 (Mobil SHC 626) |
| 냉매 회수·재충전 | 냉매 순도 확인 후 재충전 |
| 전기 계통 전반 점검 | 전선 피복, 단자 산화, 접지 저항 |
| 성능 시험 성적서 발행 | 외부 업체 서명 필수 |

## 5. 정기 점검 결과 이상 발견 시

점검 중 아래 항목이 발견되면 해당 고장 유형 대응 가이드라인으로 이관한다.

| 발견 항목 | 이관 가이드라인 |
|---|---|
| 과열 징후 (냉각 팬 이상, 오일 오염, 냉매 부족) | MNT-GUIDE-001 (Overheating) |
| 진동·소음 이상 (볼트 이완, 방진 패드 손상) | MNT-GUIDE-002 (Vibration Issue) |
| 압력 저하 (배관 누설, 밸브 이상, 필터 막힘) | MNT-GUIDE-003 (Pressure Drop) |
| 전기 이상 (인버터 에러, 절연 저항 저하) | MNT-GUIDE-004 (Electrical Fault) |
